# 04.有序数组的Java代码
有序数组的两个特点：
  1) 插入之前进行比较，按照顺序，比较次数和移动的数据时间之和为O(N)
  2) 查找的时候可以使用二分法，查找速度比较快，符合对数规律

# 08.为什么不用数组表示一切？
 * 效率问题：不同需求下数组的效率不同
 * 内存使用问题：数组初始化之后内存占用大小就固定，无法根据需要变动
